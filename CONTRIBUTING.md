<a name="english"></a>
## 🇬🇧 English

Thank you for your interest in contributing to **CuRe Code**! We welcome contributions from everyone.

### How Can I Contribute?

*   **Reporting Bugs:** Open an issue and describe the bug with steps to reproduce it.
*   **Suggesting Features:** Have an idea? Open an issue to discuss it.
*   **Improving Docs:** Documentation is never finished. Feel free to fix typos or clarify sections.
*   **Submitting Code:** We love Pull Requests! Please follow our development workflow.

### Development Setup

1.  **Clone the repo:**
    ```bash
    git clone https://github.com/broman0x/cure-code.git
    cd cure-code
    ```
2.  **Install dependencies:**
    ```bash
    go mod tidy
    ```
3.  **Run the project:**
    ```bash
    go run main.go
    ```
4.  **Web Development:**
    ```bash
    cd web
    npm install
    npm run dev
    ```

### Pull Request Process

1.  Fork the repository and create your branch from `main`.
2.  Ensure your code follows Go standards and passes linting.
3.  Update documentation if you've added/changed functionality.
4.  Submit your PR and wait for review!
